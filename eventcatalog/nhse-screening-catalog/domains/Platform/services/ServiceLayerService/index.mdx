## Overview

The Sevice Layer acts as the egress and ingress for the National Screening Platform. It provides a link with the legacy products and provides an anti-corruption layer so that within the NSP the data is consistent

## Responsibilities

1. **Anti corruption**

   - Validate inbound data requests
   - Map to internal events, that are consistent with the target data model

2. **Provied external access**
   - Make data available for external consumption through FHIR APIs
