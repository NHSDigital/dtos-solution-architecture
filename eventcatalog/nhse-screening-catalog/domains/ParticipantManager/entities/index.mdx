## Overview

This section documents all entities within the Participant Manager domain, their relationships, and how they interact with each other.

## Domain Entities

### Core Entities

1. [Participant](./Participant) - The aggregate root representing a person in the screening system
2. [PathwayEnrollment](./PathwayEnrollment) - Represents the relationship between a Participant and a Pathway
3. [Episode](./Episode) - Represents a period of screening activity

### Reference Entities

1. [Pathway](./Pathway) - Represents a screening program with its rules and settings
