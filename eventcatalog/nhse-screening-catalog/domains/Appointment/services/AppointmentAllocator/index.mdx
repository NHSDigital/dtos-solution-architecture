## Overview

The Appointment Allocator service is responsible for managing the allocation of screening appointments across different locations. It optimizes the use of available capacity while ensuring efficient resource utilisation and participant satisfaction.

## Responsibilities

1. **Capacity Management**

   - Monitor available screening slots
   - Track resource utilization
   - Handle capacity constraints
   - Optimize slot allocation

2. **Appointment Allocation**

   - Process appointment requests
   - Match requests with available capacity
   - Consider participant preferences
   - Handle priority cases

3. **Resource Optimization**
   - Balance workload across locations
   - Optimize staff utilization
   - Manage equipment availability
   - Handle facility constraints

## Service Interactions

### Receives Events

- `AppointmentRequested`: When a new appointment is needed
- `CapacityUpdated`: When screening capacity changes
- `AppointmentCancelled`: When an appointment is cancelled

### Sends Events

- `AppointmentAllocated`: When an appointment slot is allocated
- `CapacityWarning`: When capacity constraints are detected

## Business Rules

1. Appointments must be allocated within pathway-specific timeframes
2. Capacity must be optimized across all locations
3. Priority cases must be handled appropriately
4. Resource utilization must be balanced
