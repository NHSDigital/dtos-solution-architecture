## Overview

The Appointment Booker service is responsible for managing the booking process of screening appointments. It ensures that appointments are booked in accordance with pathway requirements, participant preferences, and system constraints.

## Responsibilities

1. **Appointment Booking**

   - Process appointment bookings
   - Validate booking requirements
   - Handle booking confirmations
   - Manage booking notifications

2. **Appointment Modifications**

   - Process modification requests
   - Handle rescheduling
   - Manage booking changes
   - Update related systems

3. **Appointment Cancellations**
   - Process cancellation requests
   - Handle cancellation notifications
   - Manage cancellation workflows
   - Update related records

## Service Interactions

### Receives Events

- `AppointmentAllocated`: When an appointment slot is allocated
- `AppointmentModificationRequested`: When an appointment needs to be modified
- `AppointmentCancellationRequested`: When an appointment needs to be cancelled

### Sends Events

- `AppointmentBooked`: When an appointment is confirmed
- `AppointmentModified`: When an appointment is changed
- `AppointmentCancelled`: When an appointment is cancelled

## Business Rules

1. Appointments must be booked within allocated timeframes
2. Modifications must respect pathway requirements
3. Cancellations must be handled appropriately
4. All changes must be properly communicated
