## Overview

The AppointmentBooker service is responsible for managing the booking process of screening appointments. It handles participant confirmations, manages appointment preferences, and ensures all booking requirements are met.

## Responsibilities

1. **Booking Management**

   - Process appointment bookings
   - Handle booking confirmations
   - Manage booking modifications
   - Track booking status

2. **Appointment Cancellations**
   - Process cancellation requests
   - Handle cancellation notifications
   - Manage cancellation workflows
   - Update related records

## Service Interactions

### Receives Events

- `BookAppointment`: Instruction to book an appointment
- `CancelAppointment`: When an appointment needs to be AppointmentCancelled

### Sends Events

- `AppointmentBooked`: When an appointment is confirmed
- `AppointmentCancelled`: When an appointment is cancelled
- `AppointmentAttended`: When an appointment is attended
- `AppointmentNotAttended`: When an appointment is not attended

## Business Rules

1. Appointments must be booked within allocated timeframes
2. Modifications must respect pathway requirements
3. Cancellations must be handled appropriately
4. All changes must be properly communicated
